# Sub-recipe: US-IRS-notice

For IRS (or US state revenue department) notices, typically tied to the
Wyoming LLC's US federal filings (Form 1120, 5472, 1040-NR, employment tax,
etc.) or its EIN.

## How to read it

- Find the notice number (top-right corner, e.g. "CP2000", "CP504", "LT11",
  "Letter 226-J", "Letter 5071C"). This drives everything else.
- Find the "respond by" or "pay by" date. IRS notices almost always state one.
- Note the tax year and form referenced.
- Note whether it's from the IRS directly or a state department of revenue —
  treat state notices the same way but say so in "Item type".

## Common notice types → meaning and urgency

| Notice | What it means | Typical urgency |
|---|---|---|
| CP2000 | Proposed change from an income/document mismatch (e.g., 1099 not on return) | 🟡 Moderate — response window usually 30 days, but ignoring it converts to a bill |
| CP14 | Balance due, first notice | 🟡 Moderate — pay or dispute before it escalates |
| CP501/CP503 | Reminder notices, balance still due | 🟡 Moderate, escalating each round |
| CP504 | Notice of intent to levy (state) | 🔴 Critical |
| LT11 / Letter 1058 | Final Notice of Intent to Levy + right to a hearing | 🔴 Critical — strict 30-day window to request a Collection Due Process hearing |
| 5071C / 4883C | Identity verification required before return processes | 🟡 Moderate — refund/processing is stalled until resolved, no penalty clock |
| Letter 226-J | Proposed Employer Shared Responsibility Payment (ACA) | 🔴 Critical — short response window, real dollar exposure |
| CP2100/CP2100A | Backup withholding notice re: mismatched TIN on a 1099 filed | 🟡 Moderate |
| Informational notices (no action requested) | FYI only | 🟢 Low |

If the notice number isn't in this list, describe what the notice says it
wants (pay, respond, verify, correct) and infer urgency from whether it
threatens levy/lien/dissolution (critical), demands a response within a
stated window (moderate), or is purely informational (low).

## Next action patterns

- **Balance-due notices (CP14/CP501/CP503/CP504)**: next action is "confirm
  whether the balance is accurate, then pay or set up a payment plan before
  the stated date — do not let it reach CP504/levy stage."
- **CP2000**: next action is "compare the notice's proposed changes against
  the actual return and prior-year records, then respond agree/disagree by
  the deadline."
- **LT11/Letter 1058**: next action is "decide within the stated window
  whether to request a Collection Due Process hearing (Form 12153) — this
  is the last appeal step before levy."
- **Identity verification (5071C/4883C)**: next action is "complete IRS
  identity verification (online or by phone) using the notice's reference
  number."
- **226-J**: next action is "review the proposed payment against employee-
  level ACA offer/coverage data and respond by the deadline — this usually
  needs a tax professional."

## Documents/info typically needed

- The notice itself (full pages, including any response form/voucher).
- EIN and entity name as shown on the notice.
- The tax return/form year referenced.
- Any prior correspondence on the same notice number.
- If disputing: supporting records (1099s, payroll records, prior filings).
